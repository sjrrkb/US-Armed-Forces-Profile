# US-Armed-Forces-Profile
profile using statistical analysis

5 questions were analyzed to develop a profile for potential military recruits.

1. Size of the potential recruits
2. Demographics of potential recruits
3. How potential recruits differ from recruits and non-recruits
  -demographics, values, importance in various domains of life, trustworthiness, view of governemnt, ability to follow rules
4. Gender differences among potential recruits in all of question 3 domains except demographics
5. Racial differences among potential recruits in all of question 3 domains except demographics
